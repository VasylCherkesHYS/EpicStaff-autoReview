Archives are extracted after saving, structure is preserved; unsupported ones are not extracted.

All supported file formats

ALLOWED
Documents
PDF, DOCX, TXT

Web
HTML
Archives
ZIP, TAR, TAR.GZ, TAR.BZ2, TAR.XZ
ARCHIVE RULES
  • Archives are extracted after saving; folder structure is preserved.
  • Password-protected archives are not extracted and will be rejected.
  • Archives containing password-protected files inside are rejected.
  • Archives containing disallowed files (e.g. .exe, .dll) are rejected entirely.
  • Unsupported archive formats (.rar, .7z) can not be uploaded.
